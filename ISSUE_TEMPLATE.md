## Template Usage Guidelines

### For Bug Reports
- Be specific about the steps to reproduce
- Include error messages and logs
- Describe the expected vs actual behavior
- Provide system/environment information

### For Feature Requests
- Explain the problem you're trying to solve
- Describe your proposed solution
- Consider alternative approaches
- Provide use cases and examples

### For Documentation Issues
- Specify which documentation is unclear
- Suggest improvements
- Provide examples of what's missing

### For Performance Issues
- Include performance metrics
- Describe the performance impact
- Provide profiling information if available
- Compare with expected performance

### For Security Issues
- **DO NOT** include sensitive information
- Follow responsible disclosure practices
- Use the security reporting process instead of public issues
- Contact security team directly for sensitive issues
